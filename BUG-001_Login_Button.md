# 🐞 BUG-001

## Title

Login button remains disabled after entering valid credentials.

---

## Environment

- Browser: Google Chrome
- Version: Latest
- OS: Windows 11

---

## Preconditions

- User account exists.
- Internet connection is available.

---

## Steps to Reproduce

1. Open the Login page.
2. Enter a valid email address.
3. Enter a valid password.
4. Click on the Login button.

---

## Expected Result

The user should successfully log into the application and be redirected to the Dashboard.

---

## Actual Result

The Login button remains disabled and the user cannot access the application.

---

## Severity

High

---

## Priority

High

---

## Status

Open
